### 工程名称：
vCenter
### 管理员：
顾泽宇（guzeyu13@otcaix.iscas.ac.cn）
### 描述：
vCenter是实验室内部IaaS客户端。
### 使用方法：
1. 配置JDK；
2. 设置Eclipse编码格式为GBK（建议使用一个新的Eclipse Workspace，避免更改编码影响到其他项目）；
3. 打开OnceCenter_win_64.product，在“Overview”选项卡左下方找到“Launch...”按钮。
